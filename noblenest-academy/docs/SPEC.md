# Noble Nest Academy — v1 Product Spec

**Status:** Source of truth for the v1 launch. Anything not in this document is OUT of scope for v1.
**Owner:** Manazeer Rahman Alvi (operator).
**Last updated:** 2026-05-16 (Phase 0).

---

## Audience

- Parents of children **0–6**.
- Children **7–10**.

## Curriculum

- **≥1,200 activities** across six 0–6 stages.
- **7–10 STEM track:** block coding → Python, robotics, AI / data literacy.
- **IQ + personality assessment** at age 9–10 producing a parent-facing **career-guidance report**.

## Languages (8)

English, French, Russian, Mandarin, Spanish, Korean, **Urdu, Arabic**.

- Urdu + Arabic require **full RTL UI**, RTL tracing stroke order, right-aligned typography.

## Pedagogy

Blended into every age tier:

- **Japanese** — empathy / community.
- **Chinese** — calligraphy / pinyin / structured arts.
- **Scandinavian** — outdoor / nature / democratic circle time / mindfulness.

## AI content

- Talking-avatar videos (Synthesia / HeyGen / Pictory-class provider) in all 8 languages with subtitles + transcripts.
- AI translation + voice cloning.
- Interactive overlay: quizzes, HTML5 canvas tracing, Web Speech API pronunciation, drawing tools.

## Roles

`parent`, `child`, `admin`, `school_admin` (institutional licensing).

## Payments

- Stripe + PayPal + at least one local method.
- Plans: **Freemium**, **Individual $10–15**, **Family $20–30** (up to 3 children), **Lifetime / Annual**, **Institutional bulk seats**.
- **Regional PPP** pricing.

## Compliance

- **COPPA** — verified parental consent + audit log.
- **GDPR** — export + erase.
- **WCAG 2.1 AA**.
- Mobile-first, **offline PWA**.

## Accreditation

Pack must be produced for **Cognia / BAC** submission.

---

## Out of scope for v1 (kill list)

- Maternal health / contraindication matrix / newborn flows.
- Teacher marketplace (independent teachers selling courses).
- Practitioner vetting.
- Payouts.
- Scholarships.
- Share cards.
- Daily.co live classes.
- Referrals.
- School inquiries (only minimal version for institutional licensing in Phase 7).

Anything matching the above MUST be deleted (not gated) in Phase 1.
